# Калькулятор

Новый Год уже на пороге, хо-хо-хо. Калькулятор с дизайном в стиле глинтвейна спешит поднять ваше настроение. Проект сочетает в себе функциональность стандартного калькулятора с эстетически приятным визуальным оформлением, которым хочется наслаждаться.

## Особенности реализации

- **Интуитивный интерфейс** - стандартное расположение кнопок для удобства использования
- **Тематический дизайн** - цветовая схема, вдохновленная глинтвейном (бордовые, золотые и коричневые тона)
- **Фоновое изображение** - поддержка пользовательских фоновых изображений
- **Двойной ввод** - поддержка как кликов мышью, так и ввода с клавиатуры
- **Обработка ошибок** - валидация ввода и обработка исключительных ситуаций
- **Кроссплатформенность** - работает на Windows, macOS и Linux

## Функциональность

### Основные операции и соответствующие им кнопки
- Сложение, вычитание, умножение, деление
- Вычисление процентов
- Смена знака числа
- Извлечение квадратного корня
- Операции с плавающей точкой

### Управление
- **C** - полная очистка
- **⌫** - удаление последнего символа
- **±** - смена знака числа
- **%** - вычисление процента

## Технические требования

- Python 3.8+
- Библиотека Pillow

## Установка и запуск

1. Клонируйте репозиторий:
bash
git clone https://github.com/username/cocktail-calculator.git
cd gluhwein-calculator
2. Установите зависимости:
pip install -r requirements.txt
3. Убедитесь, что файл glintwein.jpg находится в корневой директории проекта
4. Запустите приложение:
python winter_calculator.py
